# 🧪 Poison Detection 
 
This application predicts probable toxic substances based on observed symptoms, exposure route, odor, and forensic evidence.

## 🚀 Features

- Predict probable toxic substances
- Clinical symptom analysis
- Exposure route selection
- Odour-based indicators
- Timeline assessment
- Sample recommendation support
- Streamlit interactive UI
- Machine Learning model integration

---

## 🛠 Tech Stack

- Python
- Streamlit
- Scikit-learn
- XGBoost
- CatBoost
- Pandas
- NumPy
- Joblib
- Matplotlib
- Seaborn

---



## ⚙ Installation

Clone repository:

```bash
git clone https://github.com/somyaaa07/poise-setection.git
cd poise-setection/frontend
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run application:

```bash
python -m streamlit run app.py
```

---

## 📊 Model Training

Train model:

```bash
python train_toxic_model.py
```

Generated model:

```txt
model/toxic_detection_model.pkl
```
